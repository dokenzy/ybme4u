YBM e4u EngClass Crawler
========================

About
-----
 * Scrapy 공부용
 * Python 2.7
 * 첫페이지에 링크된 11개의 문제에 대한 정보를 가져오도록 함


Usage
-----

 * scrapy crawl ybme4u
 * scrapy crawl ybme4u -o result.csv -t csv
 * scrapy crawl ybme4u -o result.xml -t xml
 * scrapy crawl ybme4u -o result.json -t json
